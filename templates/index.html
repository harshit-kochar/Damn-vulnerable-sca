Welcome to SCAGoat

SCAGoat is a vulnerable web application that is designed to help security enthusiasts, developers, and students to discover and mitigate security vulnerabilities in their applications. It is a collection of security vulnerabilities and misconfigurations that can be used to test the security tools and applications.

SCAGoat is designed to be used as a learning tool and is not recommended to be used in a production environment.
